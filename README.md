# Windows専用のdotfiles
[!NOTE]
Windowsで必要な設定ファイルのみを置いている。

メインリポジトリは[ここ](https://github.com/fura0402/dotfiles)

## How to Install.

[!WARN]
Full Installのみサポート

#### 1. Microsoft StoreからPowershellをインストール

#### 2. 以下を実行
```pwsh
. env:USERPROFILE\dotfiles\dot_win\installer\install.ps1
```

#### 3. 手動でインストールが必要なアプリ
* Adobe製品
